Introduction to citoMove

Thibault Fronville & Maximilian Pichler

March 05 2026

Abstract

‘citoMove’ allows you to prepare movement data (including biotic and abiotic interactions) and build neural networks (NN) using the ‘cito’ package

First steps

Assuming you have citoMove installed (for instructions, see here), load the citoMove package via

library(citoMove)

In case you have installation problems of citoMove, please follow the instructions given in the cito package vignette

Example analysis using the amt::deer dataset

set.seed(123)
library(amt)
#> 
#> Attaching package: 'amt'
#> The following object is masked from 'package:stats':
#> 
#>     filter
library(citoMove)
library(terra)
#> terra 1.7.29

The following code creates a spatial covariate: the distance to the next forest.

forest <- amt::get_sh_forest()
forest <- terra::subst(forest, 0, NA)
forest_dist <- distance(forest)
names(forest_dist) <- "forest_dist"
plot(forest_dist)

Next, we load the tracking data of one red deer individual. From amt documentation: 826 GPS relocations of one red deer from northern Germany. The data is already resampled to a regular time interval of 6 hours and the coordinate reference system is transformed to epsg:3035.

The follwoing code orders the data, creates random absences, and extract the environmental predictor (distance to forest) create before for all true and random steps.

dat_ssf <- amt::deer |> 
  steps_by_burst() |> 
  random_steps() |> 
  extract_covariates(forest_dist) |> 
  time_of_day() |> 
  mutate(case_ = as.integer(case_))
#> Warning in random_steps.bursted_steps_xyt(steps_by_burst(amt::deer)): Some
#> bursts contain < 3 steps and will be removed

Now, we can fit the deep neural network (dnn) using citoMove:

model = dnn_ssf(case_ ~ forest_dist, data = dat_ssf, epoch = 30L, plot = T, verbose = FALSE)
